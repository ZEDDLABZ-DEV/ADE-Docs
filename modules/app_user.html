  • getAppUsersOfSpecificRole(req: Request<{ appUserRole: string }, any, any, { age: string; date: string; gender: string; isBanned: string; labourers: string; name: string; page: string; phone: string; size: string; status: string }, Record<string, any>>, res: Response<any, Record<string, any>>): Promise<any>
    • Defined in app-user.ts:129

    All the information about a particular App users is fetched from the database and is send to the client. Pagination is included in this API. The information to be sent varies for all the three App users. For Labour Data can be ordered on the basis of age, gender or ID(desc). Data includes all the information about the labour for example where it is working and if working all the infomation about the mill. For Middleman Data can be ordered on the basis of age, number of labourers working under him/her, gender or ID(desc). Data includes all the information about the middleman for example all its workers, their information, bankdetails, their job applications, their working locations. For MillOwner Data is ordered basis of ID(desc). Data includes all the information about the millowner.
